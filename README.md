# Kasthuri Mess Website

This is a **premium, responsive, and modern website** for **Kasthuri Mess**, a popular South Indian restaurant located in **Thoothukudi, India**.  

The website showcases:

- **Home / Hero Section** with call-to-action buttons
- **About Section** describing the restaurant
- **Menu Section** with Veg & Non-Veg meals
- **Services & Features**
- **Google Reviews Section**
- **Popular Meals Section**
- **Contact & Footer**
- **Floating WhatsApp button** for quick messaging

### Built With

- HTML5
- CSS3
- Google Fonts (Playfair Display & Lato)
- Responsive Grid Layout
- Modern card hover effects
